# ShoppingWeb
온라인 애견 물품샵

AWS EC2 에 구축 완료(http://3.39.225.205/)

# 사용 기술
* Go
* React
* Mysql
* AWS EC2
* nginx-1.28.0

# Overview
- bcrypt 암호화를 사용하여 가입자의 비밀번호 저장
- go루틴을 사용하여 효율적인 스레드 관리
- 이메일 인증 방식으로 회원 가입
- Token & refresh 방식으로 로그인 유지
- 고루틴 사용으로 성능 극대화
